Ontwikkeling

Mijn ontwikkeling op vorige scholen heeft weinig te maken me de opleiding die ik nu doe (software developer).
De vooropleiding die ik deed op het VMBO vond ik leuk, ik heb daar gekozen voor PIE (produceren, installeren, elektriciteit).
Daar heb ik vooral leren metaal bewerken, maar ook een klein beetje elektriciteit aanleggen, wat ik minder vond.
Dit heeft weinig toegevoegde waarde aan mijn huidige opleiding.

Mijn ontwikkelpunten zijn onder andere duidelijkere code schrijven, en andere manieren om iets te doen te ontdekken en te gebruiken.
Na de lessen werk ik thuis uren door omdat coderen mijn hobby geworden is,
dus spendeer ik meestal mijn vrije tijd met het zo fancy mogelijk maken van mijn code en eindresultaat.
